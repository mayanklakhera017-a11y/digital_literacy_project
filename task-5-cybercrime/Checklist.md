* Use Strong, Unique Passwords: Create complex passwords using a mix of upper/lower case, numbers, and symbols for every account.



* Enable Multi-Factor Authentication (MFA): Add an extra layer of security using authenticator apps rather than just SMS codes.



* Verify URL Authenticity: Always check the address bar for the correct domain name and the "HTTPS" padlock before entering data.



* Beware of Phishing Emails: Do not click on links or download attachments from unknown senders or "urgent" security alerts.



* Keep Software Updated: Regularly install security patches for your operating system and antivirus software to fix vulnerabilities.



* Avoid Public Wi-Fi for Banking: Never conduct financial transactions on unprotected public networks; use a VPN if necessary.



* Back Up Data Regularly: Maintain encrypted backups of your important academic and personal files on an external drive.



* Practice Social Media Privacy: Limit the amount of personal information (like your phone number or location) visible on public profiles.
